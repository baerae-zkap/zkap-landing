# ZKAP Landing Page Redesign — Design Spec

**Date:** 2026-03-30
**Author:** Jongin Park
**Status:** Approved
**Deadline:** 2026-04-03 (v0.4.3 앱 심사 전)

## 1. Overview

ZKAP 랜딩 페이지(www.baerae.com)를 전면 교체한다. 기존 크립토 투자 플랫폼 소개에서 **해외 가상자산 세금 신고 서비스** 중심으로 완전히 새로 구성한다.

### 목적
- v0.4.3 앱스토어 심사 전 홈페이지 업데이트 (필수)
- 해외 가상자산 신고 기능을 핵심 가치로 전달
- 앱 다운로드 전환율 극대화

### 타겟 사용자
- 해외 거래소(Binance, Bybit, OKX, Coinbase)에 자산을 보유한 한국인
- 해외 금융자산 5억원 이상 보유로 신고 의무가 있는 투자자
- 해외 거주 한국인 포함

## 2. Tech Stack

| 항목 | 선택 |
|------|------|
| Framework | Next.js (App Router) |
| Styling | Tailwind CSS |
| Animation | Framer Motion + 스크롤 프레임 시퀀스 |
| i18n | next-intl 또는 i18next (한국어/English) |
| Analytics | GA4 (기존 ID: G-2TB2G9PKB0) |
| Deployment | 인프라팀 요청 (빌드 산출물 전달) |

## 3. Design Direction

**톤앤매너:** 클린 & 신뢰감 (토스/뱅크샐러드 스타일)
- 밝은 배경 (#f8fafc ~ #fff)
- 깔끔한 타이포그래피
- 서브틀 그리드 패턴
- 인디고/바이올렛 포인트 컬러 (#4f46e5)
- 경고 구간은 레드/오렌지 포인트

**인터랙션:**
- 각 섹션 스크롤 진입 시 fade-in + slide-up (Framer Motion)
- 프레임 시퀀스 스크롤 애니메이션 (Apple 제품 분해 스타일)
- 신고 프로세스 스텝별 순차 등장
- FAQ 아코디언 토글

## 4. Psychological Narrative Arc

기능 나열(Feature-driven)이 아닌 심리적 설득 서사(Solution-driven) 구조:

```
😰 공포/자각 → 🤔 개인화 → 😮‍💨 안도 → ✅ 해결책 → 🛡️ 신뢰 → 📲 행동
```

## 5. Section Specification

### Section 1: Hero — 손실 회피형 후킹
- **심리:** 공포, 자각, 손실 회피
- **카피(KO):** "해외 거래소 자산, '몰랐다'는 통하지 않습니다"
- **보조:** "5억 이상 해외 금융자산, 신고는 의무입니다"
- **CTA:** App Store / Google Play 다운로드 버튼
- **비주얼:** 밝은 배경 + 서브틀 그리드. 앱 목업 이미지 (교체 가능)
- **애니메이션:** 텍스트 staggered fade-in, CTA 부드러운 등장

### Section 2: 위험 경고 — 구체적 리스크
- **심리:** 긴박감, 공포 구체화
- **내용:**
  - 미신고 시 과태료 최대 20%
  - 2년 이하 징역 또는 2억원 이하 벌금
  - 고정 예시 카드: "10억 미신고 → 과태료 최대 2억"
  - 법적 근거 출처 명시 (국제조세조정법)
- **비주얼:** 경고 카드 UI, 레드/오렌지 포인트, 숫자 카운트업 애니메이션

### Section 3: 자가진단 — "혹시 나도 대상?"
- **심리:** 개인화, 몰입, 자기 관련성
- **내용:** 시각적 체크리스트 (인터랙티브 폼이 아닌 스크롤 애니메이션)
  - ☑ 해외 거래소에 자산을 보유하고 있다
  - ☑ 작년 중 자산 총액이 5억을 넘은 적이 있다
  - ☑ 여러 거래소에 분산되어 총액 파악이 어렵다
  - → "하나라도 해당되면, 지금 확인이 필요합니다"
- **애니메이션:** 스크롤 시 체크박스가 하나씩 체크되는 효과

### Section 4: 안도감 전환 — "ZKAP이면 간단합니다"
- **심리:** 안도, 감정 전환 (공포 → 안심)
- **카피(KO):** "복잡한 해외 자산 신고, ZKAP 하나면 충분합니다"
- **보조:** "흩어진 거래소 자산을 모아보고, 전문가와 신고까지 한 번에"
- **비주얼:** 큰 타이포그래피, 그린/블루 톤 전환, 심플하게

### Section 5: 거래소 연동 + 자산 한눈에 (통합)
- **심리:** 해결책 제시, 편리함
- **내용:**
  - Binance, Bybit, OKX, Coinbase 로고 배치
  - "거래소 계정 연결만 하면 끝"
  - 연동 후 → 거래소별 보유 자산, 월말 잔고, 원화 환산 대시보드
  - 연동과 조회를 하나의 스토리로 연결
- **비주얼:** 거래소 로고 카드 → 대시보드 UI 전환
- **애니메이션:** 스크롤 프레임 시퀀스 (앱 화면 전환). 목업 이미지로 구현, 나중에 실제 스크린샷으로 교체 가능하도록 이미지 배열 관리
- **기술 참고:** 거래소 연동은 웹뷰 로그인 방식 (API 키 아님). 랜딩에서는 기술 방식을 드러내지 않음

### Section 6: 신고 프로세스 3스텝
- **심리:** 행동 장벽 제거, 간편함
- **내용:**
  1. 거래소 연결 (로그인 한 번)
  2. 자산 확인 (한눈에 파악)
  3. 전문가와 신고 (세무사가 대행)
- **애니메이션:** 스텝 순차 등장 + 연결선 그려지는 효과

### Section 7: 세무사 매칭 + 신뢰 증거
- **심리:** 권위, 신뢰, 불안 해소
- **내용:**
  - 전문 세무사가 신고를 대행
  - 보안 인증 배지, 개인정보 보호 강조
  - 이용자 수/후기 (데이터 있을 경우)
- **비주얼:** 세무사 아이콘/일러스트 + 신뢰 배지 그리드

### Section 8: FAQ — 망설임 제거
- **심리:** 이의 처리, 최종 의구심 해소
- **형태:** 아코디언
- **예상 항목:**
  - "신고 대상인지 어떻게 알 수 있나요?"
  - "어떤 거래소를 지원하나요?"
  - "수수료는 얼마인가요?"
  - "개인정보는 안전한가요?"
  - "신고 기한은 언제까지인가요?"

### Section 9: Final CTA + Footer
- **심리:** 행동 유도, 마지막 푸시
- **CTA 카피(KO):** "과태료 걱정 없는 가장 쉬운 방법, 지금 시작하세요"
- **CTA:** 앱 다운로드 버튼 반복 (App Store / Google Play)
- **Footer:**
  - Baerae Co., Ltd. 회사 정보
  - 개인정보처리방침 / 이용약관 (외부 링크)
  - SNS 링크 (X, Medium)
  - Copyright © 2026

## 6. Responsive Design

- Mobile-first 접근
- Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
- 모바일에서도 스크롤 프레임 애니메이션 동작 (성능 고려하여 프레임 수 조정 가능)

## 7. i18n Strategy

- 한국어 (기본, fallback)
- English
- next-intl 또는 i18next로 번역 파일 분리
- 헤더에 언어 토글 버튼

## 8. Image/Asset Strategy

- 스크롤 프레임 시퀀스: 목업 이미지 배열로 구현 → 실제 스크린샷으로 교체 가능
- 거래소 로고: Binance, Bybit, OKX, Coinbase SVG
- 앱 스토어 배지: App Store, Google Play 공식 배지
- 일러스트/아이콘: 세무사, 보안, 체크리스트 등 (SVG 또는 Lucide icons)

## 9. Migration Notes

- 기존 CRA 레포에서 Next.js로 전면 교체 (새로 구축)
- 기존 뉴스레터 구독 기능 (Google Apps Script) 제거 → 앱 다운로드로 대체
- GA4 추적 ID 유지 (G-2TB2G9PKB0)
- CNAME (www.baerae.com) 도메인은 인프라팀과 조율

## 10. Out of Scope

- 개인정보처리방침/이용약관 페이지 자체 구현 (외부 링크로 연결)
- 실제 세금 계산 시뮬레이터 (법적 리스크)
- 인터랙티브 자가진단 폼 (시각적 체크리스트로 대체)
- 실시간 앱 데이터 연동
